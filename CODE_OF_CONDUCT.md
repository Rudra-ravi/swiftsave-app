# Code of Conduct

## Our standards

We expect all participants to:

- Be respectful and professional.
- Assume good intent and communicate clearly.
- Accept constructive feedback.
- Focus discussion on the code and the problem.

Unacceptable behavior includes harassment, personal attacks, hate speech,
doxxing, or any behavior that makes participation unsafe.

## Enforcement

Project maintainers may remove comments, close discussions, or block users for
violations.

## Reporting

Report conduct concerns to the maintainers via private security/contact channel
defined in `SECURITY.md`.
