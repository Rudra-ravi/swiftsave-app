## 0.0.1

* Initial release of ytdlp_bridge plugin
* Features:
  - Video information extraction via yt-dlp
  - Multi-format video download support
  - Playlist detection and handling
  - Progress callback support for downloads
  - Support for audio-only downloads with automatic format selection
  - Integration with Chaquopy for Python runtime on Android
  - Optimized for mobile with concurrent fragment downloads (4 fragments)
* Platform support: Android (arm64-v8a, x86_64)
* Python version: 3.12
* yt-dlp version: 2025.12.08
